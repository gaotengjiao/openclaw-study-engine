# study-ingest Skill

知识输入与结构化技能 - StudyEngine 核心模块

## 功能

- 接收用户输入（文本/语音/代码片段/文档）
- AI 自动补全 2-3 级知识结构
- 敏感信息检测与脱敏
- 存入 Neo4j 知识图谱 + SQLite 进度库
- 支持断点续传

## 触发方式

用户主动调用，支持以下输入格式：

- `学习 [概念名]` - 开始学习新概念
- `继续` - 恢复上次学习进度
- `导入 [文件路径]` - 导入文档/代码

## 配置

在 `config.yaml` 中配置：

- Neo4j 连接信息
- SQLite 数据库路径
- 敏感信息检测规则
- GitHub 发布配置（可选）

## 依赖

- `@openclaw/storage-neo4j` 插件
- Python 3.8+
- `pymongo` (可选，用于 MongoDB 备份)

## 输出

- 知识图谱构建完成通知
- 薄弱点初始评估
- 下一步学习建议
